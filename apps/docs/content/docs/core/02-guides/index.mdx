## Quick Reference

Common patterns at a glance:

```ts
// Seek to time
core.execute({ id: nanoid(), type: "playback.seek", payload: { time: 5_000_000 } });

// Add clip
core.execute({ id: nanoid(), type: "clip.add", payload: { trackId: "main", clip } });

// Move clip
core.execute({ id: nanoid(), type: "clip.move", payload: { clipId: "clip_1", timing: {...} } });

// Batch operations
core.batch([cmd1, cmd2, cmd3]);

// Undo/redo
core.undo();
core.redo();
```
