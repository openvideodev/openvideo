## Quick Links

### Core Methods

| Method                | Description               |
| --------------------- | ------------------------- |
| `core.execute(cmd)`   | Execute a single command  |
| `core.batch(cmds)`    | Execute multiple commands |
| `core.undo()`         | Undo last action          |
| `core.redo()`         | Redo last undone action   |
| `core.reset(project)` | Load new project          |

### Store Access

```ts
// Get current state
const state = core.store.getState();

// Subscribe to changes
const unsubscribe = core.store.subscribe((state, prev) => {
  console.log("State changed");
});

// Listen to patches
const off = state.onChange((patches) => {
  console.log("Patches:", patches);
});
```

### Playback

```ts
core.playback.play();
core.playback.pause();
core.playback.toggle();
core.playback.seek(time);
```

## Common Patterns

### Command Structure

```ts
{
  id: string;           // Unique identifier (use nanoid())
  type: string;         // Command type (e.g., "clip.add")
  payload: object;      // Command-specific data
  meta?: {
    source?: "user" | "agent" | "system";
    timestamp?: number;
    userId?: string;
  };
}
```

### Patch Structure

```ts
{
  op: "add" | "update" | "remove";
  path: string;         // JSON Pointer path
  value?: any;          // New value
  oldValue?: any;       // Previous value (for undo)
}
```
