Effects allow you to apply visual filters and modifications to clips.

## Applying Effects

Effects are added by pushing to the `effects` array on a clip.

```ts
import { Video } from "openvideo";

const clip = await Video.fromUrl("video.mp4");

// Add a sepia effect
clip.effects.push({
  id: "my-effect",
  key: "sepia",
  startTime: 0,
  duration: 5e6, // 5 seconds in microseconds
});
```

## Supported Effect Keys

OpenVideo supports a wide range of visual effects, categorized into built-in GLSL effects and specialized Pixi Filters.

### Pixi Filters

These are specialized filters provided by the `pixi-filters` library. Note that some keys end in `Filter` to avoid collisions.

- `adjustmentFilter`: Gamma, saturation, contrast, brightness, and color channel adjustments.
- `advancedBloomFilter`: High-quality glow/bloom control.
- `asciiFilter`: Converts the image to ASCII characters.
- `backdropBlurFilter`: Blurs everything behind the clip (useful for overlays).
- `bevelFilter`: Adds a 3D bevel/shadow effect to the edges.
- `bloomFilter`: Standard blurring glow.
- `bulgePinchFilter`: Bulge or pinch distortion at a specific point.
- `colorGradientFilter`: Advanced multi-stop color gradients.
- `colorMapFilter`: Uses a texture map for color lookup.
- `colorOverlayFilter`: Simple solid color tint.
- `colorReplaceFilter`: Replaces a specific color with another.
- `convolutionFilter`: Generic 3x3 convolution matrix (sharpen, emboss, etc.).
- `crossHatchFilter`: Stylized cross-hatch drawing effect.
- `crtFilter`: Comprehensive old-school monitor simulation (noise, curvature, scanlines).
- `dotFilter`: Halftone dot pattern.
- `dropShadowFilter`: Adds a drop shadow to the clip.
- `embossFilter`: 3D embossed texture effect.
- `glitchFilter`: Intense digital glitching with slices and offsets.
- `glowFilter`: Configurable outer and inner glow.
- `godrayFilter`: Volumetric light ray effect.
- `grayscaleFilter`: Simple grayscale filter.
- `hslAdjustmentFilter`: Hue, saturation, and lightness control.
- `kawaseBlurFilter`: High-performance blur.
- `motionBlurFilter`: Directional blur based on velocity.
- `multiColorReplaceFilter`: Batch color replacement.
- `oldFilmFilter`: Scratches, noise, and sepia for a vintage look.
- `outlineFilter`: Adds a solid outline around the clip content.
- `pixelateFilter`: Standard blocky pixelation.
- `radialBlurFilter`: Blur that radiates from a specific center point.
- `reflectionFilter`: Dynamic water-like reflection with waves.
- `rgbSplitFilter`: Standard RGB channel splitting.
- `shockwaveFilter`: Expanding shockwave ripple animation.
- `simpleLightmapFilter`: Uses a lightmap texture for lighting.
- `simplexNoiseFilter`: Generates and applies simplex noise.
- `tiltShiftFilter`: Simulates a tilt-shift camera lens (partial blur).
- `twistFilter`: Twists the image around a center point.
- `zoomBlurFilter`: Blur that zooms in/out from a center point.

## Using Effect Values

Many effects allow you to customize their appearance by passing a `values` object.

```ts
clip.effects.push({
  id: "custom-glow",
  key: "glowFilter",
  startTime: 0,
  duration: 5e6,
  values: {
    distance: 20,
    outerStrength: 3,
    color: "#ff0000",
  },
});
```

### Animating Values

While built-in effects often handle animations internally (using `uTime`), you can also manually update values or use the internal animation system for complex transitions.

## Managing Effects

Since `effects` is a standard array, you can use standard array methods:

```ts
// Remove an effect
clip.effects = clip.effects.filter((e) => e.id !== "my-effect");

// Clear all effects
clip.effects = [];
```

## Custom Effects

You can add your own custom effect shaders using the `registerCustomEffect` function. This allows you to extend the engine with your own GLSL filters.

### Registering a Custom Effect

To register a custom effect, you need to provide a unique key and an effect object containing the GLSL fragment shader.

```ts
import { registerCustomEffect } from "openvideo";

registerCustomEffect("myCustomEffect", {
  label: "My Custom Effect",
  fragment: `
    vec4 mainImage(vec2 p) {
      vec4 color = getTextureColor(p);
      return vec4(color.g, color.r, color.b, color.a); // Swap R and G channels
    }
  `,
});
```

### Usage in Editor/Studio

Once registered, you can use the effect key just like any built-in effect:

```ts
clip.effects.push({
  id: "custom-1",
  key: "myCustomEffect",
  startTime: 0,
  duration: 5e6,
});
```

### GlEffect Interface

The effect object follows this structure:

| Property           | Type     | Description                                                  |
| :----------------- | :------- | :----------------------------------------------------------- |
| **label**          | `string` | Human-readable name for the effect.                          |
| **fragment**       | `string` | GLSL fragment shader code (main entry point is `mainImage`). |
| **uniforms**       | `Record` | (Optional) Default values for shader uniforms.               |
| **previewStatic**  | `string` | (Optional) URL for a static preview image.                   |
| **previewDynamic** | `string` | (Optional) URL for a dynamic preview image or video.         |
