# podcast-edit

## Overview
Optimizes podcast recordings for clarity and professionalism.

## Instructions
Use this for "podcast", "interview", or "talking head" videos.

### Implementation Details
The backend will:
1. Detect and remove silence/dead air.
2. Normalize audio across all tracks.
3. (Optional) Add lower-third overlays if speaker names are provided.
