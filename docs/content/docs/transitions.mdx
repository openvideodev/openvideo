---
title: Transitions
description: Smooth visual effects between consecutive clips.
---

Transitions create smooth visual effects between two consecutive clips on the timeline. They are applied at the boundary where one clip ends and another begins.

## Usage

To add a transition, you can use the `addTransition` method on the `Studio` instance. It is recommended to add transitions after both clips have been added to the studio.

```ts
import { Studio, Video, Transition } from 'openvideo';

const studio = new Studio({
  width: 1080,
  height: 1920,
  fps: 30,
  canvas: canvasRef.current!,
});

const video1 = await Video.fromUrl('video1.mp4');
studio.addClip(video1);

const video2 = await Video.fromUrl('video2.mp4');
video2.display = {
  from: video1.duration,
  to: video1.duration + video2.duration,
};
studio.addClip(video2);

// Add transition from video1 to video2
// Usage: studio.addTransition(key, duration, fromClipId, toClipId)
try {
  await studio.addTransition('cube', 2 * 1e6, video1.id, video2.id);
  console.log('Transition added successfully');
} catch (e) {
  console.error('Failed to add transition:', e);
}
```

## Overlap Management

OpenVideo automatically handles the visual overlap during the transition. The software ensures that both clips are active and rendered during the transition duration. The transition center is aligned with the junction of the two clips.
