Enhance your videos with OpenVideo's creative tools.

## Animation

Bring clips to life with motion:

- **[Animations](./animations)** - Keyframe-based animation system
- Move, scale, rotate, fade—anything over time

## Transitions

Smooth clip transitions:

- **[Transitions](./transitions)** - Fade, slide, wipe, dissolve
- Automatic interpolation between clips

## Visual Effects

Professional video effects:

- **[Effects](./effects)** - Blur, color correction, filters

## Quick Example

```ts
// Animate position
core.execute({
  id: nanoid(),
  type: "clip.addKeyframes",
  payload: {
    clipId: "clip_1",
    property: "transform.x",
    keyframes: [
      { time: 0, value: 0, easing: "easeOut" },
      { time: 1_000_000, value: 100, easing: "linear" },
    ],
  },
});

// Add transition
core.execute({
  id: nanoid(),
  type: "transition.add",
  payload: {
    trackId: "main",
    between: ["clip_1", "clip_2"],
    type: "fade",
    duration: 500_000,
  },
});
```
