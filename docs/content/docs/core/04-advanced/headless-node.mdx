OpenVideo Core can run in Node.js for server-side processing, automation, and backend services. This guide covers setup and usage in headless environments.

## Why Headless?

- **Server-side rendering**: Generate videos on your backend
- **Batch processing**: Process multiple videos without UI
- **Automation**: CI/CD pipelines, scheduled jobs
- **Microservices**: Dedicated video processing service

## Setup

### Installation

```bash
npm install @openvideo/core
```

### Polyfills

Core requires some browser APIs that need polyfills in Node.js:

```bash
npm install ws @peculiar/webcrypto
```

```ts
// polyfills.ts
import { Crypto } from "@peculiar/webcrypto";
import { WebSocket } from "ws";

// WebCrypto
global.crypto = new Crypto();

// WebSocket (for real-time features)
global.WebSocket = WebSocket as any;

// Other globals if needed
global.fetch = fetch; // Node 18+ has native fetch
```

## Basic Usage

### Creating a Project

```ts
import { Core } from "@openvideo/core";
import "./polyfills";

const core = new Core({
  settings: {
    width: 1920,
    height: 1080,
    fps: 30,
    duration: 60_000_000,
  },
  tracks: [
    { id: "video", name: "Video", clips: [] },
    { id: "audio", name: "Audio", clips: [] },
  ],
});
```

### Loading Media from URLs

```ts
// In headless mode, src can be:
// - HTTP URL (publicly accessible)
// - File path (local filesystem)
// - Base64 data URI

const video = await core.clip.prepare({
  type: "Video",
  src: "https://example.com/video.mp4",
  // or
  // src: "file:///path/to/local/video.mp4",
});

core.execute({
  id: nanoid(),
  type: "clip.add",
  payload: { trackId: "video", clip: video },
});
```

### Server-Side Rendering

```ts
import { render } from "@openvideo/core/rendering";
import * as fs from "fs";

async function renderVideo(project: IProject, outputPath: string) {
  const blob = await render(project, {
    format: "mp4",
    fps: 30,
    quality: "high",
  });

  const buffer = Buffer.from(await blob.arrayBuffer());
  fs.writeFileSync(outputPath, buffer);

  return outputPath;
}

// Usage
const project = core.store.getState().getSnapshot();
await renderVideo(project, "./output.mp4");
```

## Express API Example

### Video Generation Endpoint

```ts
import express from "express";
import { Core, nanoid } from "@openvideo/core";
import { render } from "@openvideo/core/rendering";
import "./polyfills";

const app = express();
app.use(express.json());

// Generate video from template
app.post("/api/render", async (req, res) => {
  try {
    const { template, media, text } = req.body;

    // Create project
    const core = new Core({
      settings: { width: 1920, height: 1080, fps: 30, duration: 30_000_000 },
      tracks: [{ id: "main", name: "Main", clips: [] }],
    });

    // Add media
    for (const item of media) {
      const clip = await core.clip.prepare({
        type: item.type,
        src: item.url,
        timing: item.timing,
      });

      core.execute({
        id: nanoid(),
        type: "clip.add",
        payload: { trackId: "main", clip },
      });
    }

    // Render
    const blob = await render(core.store.getState().getSnapshot(), {
      format: "mp4",
    });

    // Stream response
    const buffer = Buffer.from(await blob.arrayBuffer());
    res.setHeader("Content-Type", "video/mp4");
    res.send(buffer);
  } catch (error) {
    res.status(500).json({ error: error.message });
  }
});

app.listen(3000, () => {
  console.log("Video service running on port 3000");
});
```

## Batch Processing

### Processing Multiple Videos

```ts
import { Core } from "@openvideo/core";
import * as path from "path";

async function batchProcess(inputs: string[], template: any) {
  const results: string[] = [];

  for (const input of inputs) {
    const core = new Core({
      settings: { width: 1920, height: 1080, fps: 30, duration: 30_000_000 },
      tracks: [{ id: "main", name: "Main", clips: [] }],
    });

    // Load input
    const clip = await core.clip.prepare({
      type: "Video",
      src: input,
    });

    core.execute({
      id: nanoid(),
      type: "clip.add",
      payload: { trackId: "main", clip },
    });

    // Apply template (add intro, outro, etc.)
    await applyTemplate(core, template);

    // Render
    const output = path.join("output", path.basename(input));
    await renderVideo(core, output);
    results.push(output);
  }

  return results;
}
```

## Limitations

### What's Different in Headless Mode

| Feature | Browser | Node.js |
|---------|---------|---------|
| Video preview | Real-time playback | Not available |
| Canvas rendering | GPU accelerated | Software fallback |
| WebGL effects | Full support | Limited |
| Audio playback | Yes | No (process only) |
| User interaction | Yes | N/A |

### Workarounds

**No Preview**: Generate frame thumbnails:

```ts
import { extractFrame } from "@openvideo/core/rendering";

async function generateThumbnail(project: IProject, time: number) {
  const canvas = await extractFrame(project, time);
  const blob = await canvas.convertToBlob({ type: "image/png" });
  return blob;
}
```

## Best Practices

### 1. Use Streams for Large Files

```ts
import { pipeline } from "stream/promises";
import { createWriteStream } from "fs";

async function streamRender(project: IProject, outputPath: string) {
  const blob = await render(project, { format: "mp4" });
  const readable = blob.stream();
  const writable = createWriteStream(outputPath);

  await pipeline(readable, writable);
}
```

### 2. Process in Chunks

```ts
async function processLargeVideo(inputPath: string) {
  // Process 30-second segments
  const SEGMENT_DURATION = 30_000_000;
  const totalDuration = await getVideoDuration(inputPath);

  for (let start = 0; start < totalDuration; start += SEGMENT_DURATION) {
    const end = Math.min(start + SEGMENT_DURATION, totalDuration);

    const core = new Core({
      settings: { width: 1920, height: 1080, fps: 30, duration: end - start },
      tracks: [],
    });

    // Load segment
    const clip = await core.clip.prepare({
      type: "Video",
      src: inputPath,
      timing: {
        trim: { from: start, to: end },
        display: { from: 0, to: end - start },
      },
    });

    // Process and save segment
    // ...
  }
}
```

### 3. Error Handling

```ts
async function safeRender(project: IProject) {
  try {
    return await render(project, { format: "mp4" });
  } catch (error) {
    if (error.message.includes("WebCodecs")) {
      console.error("WebCodecs not available. Use Chrome/Edge in headless mode.");
    }
    throw error;
  }
}
```

### 4. Resource Cleanup

```ts
class VideoProcessor {
  private core: Core;

  async dispose() {
    // Clear all clips to release file handles
    const state = this.core.store.getState();

    this.core.batch(
      state.tracks.flatMap((t) =>
        t.clips.map((id) => ({
          id: nanoid(),
          type: "clip.remove",
          payload: { clipId: id },
        }))
      )
    );
  }
}
```

## Docker Example

### Dockerfile

```dockerfile
FROM node:18-slim

# Install Chrome for headless WebCodecs
RUN apt-get update && apt-get install -y \
    chromium \
    && rm -rf /var/lib/apt/lists/*

ENV PUPPETEER_SKIP_CHROMIUM_DOWNLOAD=true
ENV PUPPETEER_EXECUTABLE_PATH=/usr/bin/chromium

WORKDIR /app
COPY package*.json ./
RUN npm install

COPY . .
EXPOSE 3000

CMD ["node", "server.js"]
```
