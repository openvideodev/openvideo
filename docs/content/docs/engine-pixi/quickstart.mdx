This guide will walk you through setting up the GPU-accelerated **Engine Pixi** to render your video editor timeline.

## Installation

Install the engine package along with its required dependencies:

  ```bash
  pnpm add @openvideo/engine-pixi pixi.js
  ```

  </Tab>

  <Tab value="yarn">

  </Tab>
</Tabs>

## Basic Usage

Here is a full integration showing how to link a timeline to your Pixi-powered player:

```ts
import { Timeline, Track, VideoClip } from '@openvideo/core';
import { PixiRenderer } from '@openvideo/engine-pixi';

// 1. Set up a core timeline
const timeline = new Timeline();
const track = new Track();
track.addClip(new VideoClip({
  id: 'video-1',
  source: '/assets/sample.mp4',
  start: 0,
  duration: 10,
}));
timeline.addTrack(track);

// 2. Query your HTML Canvas
const canvas = document.querySelector('#preview-canvas') as HTMLCanvasElement;

// 3. Initialize the PixiRenderer
const renderer = new PixiRenderer({
  timeline,
  canvas,
  width: 1280,
  height: 720,
});

// 4. Start rendering
renderer.play();
```
