# Changelog

All notable changes to the `openvideo` project will be documented in this file.

## [0.1.9] - 2026-02-15

### Added
- New `slow` easing and expanded easing functions (`Sine`, `Circ`).
- Enhanced keyframe animation system with global/per-segment easing toggle.
- `disableGlobalEasing` option for precise keyframe control.
- Support for additional animatable properties: `blur`, `brightness`, and `mirror`.

### Changed
- Corrected position property names to `x` and `y` in documentation and examples.
- Updated package export configuration for improved bundler compatibility.

### Fixed
- Fixed TypeScript type errors in keyframe animation unit tests.

## [0.1.8] - 2026-02-11

### Added
- New animation presets.
- Support for updated animation properties.

### Changed
- Updated render logic.

## [0.1.7] - 2026-02-10

### Added
- New animation presets with updated logic for better visual effects.

### Fixed
- Fixed autoscroll playhead behavior in timeline.
- Fixed ruler timeline display issues.

## [0.1.6] - 2026-02-10

### Changed
- Internal refactoring of easing logic in keyframe animations for better consistency and simplicity.

## [0.1.5] - 2026-02-09

### Added
- Added `easing` property support for animations, allowing fine-grained control over animation timing functions.

## [0.1.4] - 2026-02-09

### Changed
- Improved animation system with enhanced parameters and smoother transitions.
- Refined animation presets for better visual effects.

## [0.1.3] - 2026-02-09

### Fixed
- Fixed incorrect package release configuration.

## [0.1.2] - 2026-02-08

### Added
- Integrated additional animation effects for text and image clips.

## [0.1.1] - 2026-02-07

### Changed
- Renamed animation presets to more descriptive names (e.g., `blurSlideRightIn`, `wobbleZoomIn`, `glitchSlideOut`).

## [0.1.0] - 2026-02-07

### Added
- Integrated GSAP-based animation system for all clip types.
- New animation presets for quick motion effects.
- Floating controls for real-time animation parameter updates.
- Support for complex animation properties (position, scale, opacity, angle, duration, delay, and easing).

### Changed
- Improved effect and transition coexistence logic.
- Refactored studio state management for better performance and stability.

### Fixed
- Fixed canvas sizing issues when parent container dimension changes.
- Resolved various minor bugs in transition and property rendering.

## [0.0.2] - 2026-02-02

### Fixed
- Fixed transition rendering in studio that was displaying duplicate frames.

## [0.0.1] - 2026-02-02

### Added
- Initial major release of the `openvideo` package (previously `openvideo`).
- Integrated WebCodecs for high-performance client-side video rendering.
- Implemented a flexible `Studio` core for interactive project management.
- Support for various clip types: `Video`, `Image`, `Text`, `Audio`, and `Caption`.
- Built-in `Effect` and `Transition` systems.
- Comprehensive `SelectionManager` and `Transformer` for interactive editing.
- `HistoryManager` for robust undo/redo support.
- JSON serialization/deserialization for project persistence.

### Changed
- Refactored `updateArtboardLayout` to ensure responsive canvas behavior.
- Improved resize detection using `ResizeObserver` and `getBoundingClientRect`.
- Optimized PIXI application initialization for better container filling.

### Fixed
- Resolved issue where canvas dimensions wouldn't grow correctly with its container.
- Fixed infinite recursion loops during renderer resize events.
